## CHEMIST framework for IELTS idea generation

Use **CHEMIST** as a structured brainstorming framework for IELTS Task 2 essays:

- **C** = Convenience
- **H** = Health
- **E** = Efficiency
- **M** = Money
- **I** = Interpersonal relationships & Relaxation
- **S** = Success
- **T** = Technology

### Core method

For any topic, analyse it from **both sides**:
1. **Positive side**: what benefits or advantages it may bring
2. **Negative side**: what drawbacks, risks, or harmful consequences it may cause

Then, for each relevant CHEMIST category, further classify ideas by **context/domain**, such as:
- **in academia**
- **in work**
- **in daily life**
- **in society**
- **And more domains. Please don't be restricted to the domains mentioned above.**

### Recommended thinking path

For each topic, apply this sequence:

1. **Choose 1–3 relevant CHEMIST categories**
   - Do not force all seven categories into one essay.
   - Select only the most relevant dimensions.

2. **Split each category into positive and negative branches**
   - Example: **Success → success / failure**
   - Example: **Health → improved wellbeing / physical or mental harm**

3. **Sub-classify each branch by domain**
   - Example: **success in academia / success in work**
   - Example: **convenience in daily life / convenience in work**
   - Example: **health in academia / health in society**

4. **Generate concrete effects under each sub-category**
   - causes
   - direct consequences
   - long-term consequences
   - affected groups

5. **Express each point in formal IELTS language**
   - avoid simple labels only
   - turn each idea into an arguable sentence suitable for body paragraphs

### Practical goal

The framework is designed to help generate:
- **advantages / disadvantages**
- **causes / effects**
- **problem / solution ideas**
- **topic-specific supporting points**

### Writing principle

Prioritise **depth over quantity**:
- It is better to develop **2 strong dimensions** clearly than to mention many weak ones.
- A good body paragraph usually follows this pattern:

**CHEMIST category → positive/negative branch → domain (academia/work/daily life/society) → specific impact → broader consequence**

### Example pattern

- **Success → positive → in academia → stronger motivation and better performance → broader educational opportunities**
- **Health → negative → in work → occupational stress and burnout → lower productivity and poorer wellbeing**
- **Convenience → positive → in daily life → less time spent on routine tasks → more manageable lifestyles**
